## 🚀 Quick start
1. **Clone Template**
   ```sh  
    git clone https://github.com/gilang-as/ant-design-example.git
   ```
2. **Install Module**
   ```shell
   yarn install
   # or
   npm install
   ```
3. Run
   ```sh
   yarn start
   # or
   npm start
   ```
